# Forecast Reliability Analysis in Electricity Load Forecasting

## Project Overview
This project investigates when electricity load forecasting models become unreliable. Instead of focusing only on prediction accuracy, the study analyzes forecast error behavior across temporal categories and demand regimes.

## Dataset
Dataset: PJM Hourly Energy Consumption  
Source: https://www.kaggle.com/datasets/robikscube/hourly-energy-consumption

## Research Questions
1. Do forecast errors vary across temporal categories such as hour of day, weekday, and season?
2. Are forecast errors significantly higher during extreme electricity demand periods?
3. Does forecast reliability deteriorate during seasonal peak demand conditions?

## Repository Structure
- `data/` dataset used in the analysis
- `notebooks/` Jupyter notebooks containing analysis and code
- `reports/` milestone reports and compiled outputs
- `figures/` generated visualizations

## Tools
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
